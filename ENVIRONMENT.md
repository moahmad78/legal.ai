# Environment Variables

You must configure the following environment variables in `.env.local` for development and in Vercel for production.

```env
# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
CLERK_WEBHOOK_SECRET=whsec_...

# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGci...

# OpenAI
OPENAI_API_KEY=sk-...

# AWS S3 (for Document Storage)
AWS_ACCESS_KEY_ID=AKIA...
AWS_SECRET_ACCESS_KEY=...
AWS_REGION=us-east-1
AWS_S3_BUCKET=catalyst-ai-documents

# Resend (Email Delivery)
RESEND_API_KEY=re_...

# Razorpay (Subscriptions)
RAZORPAY_KEY_ID=rzp_test_...
RAZORPAY_KEY_SECRET=...
RAZORPAY_WEBHOOK_SECRET=your_custom_webhook_secret
RAZORPAY_PRO_PLAN_ID=plan_...
RAZORPAY_BUSINESS_PLAN_ID=plan_...

# Sentry (Error Monitoring)
NEXT_PUBLIC_SENTRY_DSN=https://...
SENTRY_ORG=...
SENTRY_PROJECT=...

# Application
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
